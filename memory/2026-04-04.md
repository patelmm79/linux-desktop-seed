
## DNA Mechanics Mode (elastica app)

Added DNA mechanics analysis to the elastica Cloud Run app. Deployed at:
https://elastica-app-665374072631.us-central1.run.app

### What was built
- `src/dna_mechanics.py` — DNA physics module with:
  - WLC (Worm-Like Chain) force-extension model (Marko-Siggia interpolation)
  - Elastica-to-DNA unit converter (inches/lbf → nm/pN)
  - DNA material presets: B-form, A-form, nucleosomal, ssDNA
  - Euler buckling analysis for DNA
  - Supercoiling density (σ) computation
  - Loop formation free energy
  - Viral packaging force estimation
- `src/plotting_dna.py` — DNA-specific Plotly charts:
  - WLC force-extension curve
  - DNA elastica deflection shape
  - Buckling phase diagram
  - Supercoiling torque plot
  - DNA looping energy landscape
  - Summary dashboard
- `app/ui_streamlit.py` — Mode toggle (Engineering / DNA Mechanics):
  - DNA mode replaces engineering params with DNA-specific ones (µm, nm, pN)
  - B-form DNA selected by default, persistence length 50nm
  - DNA-specific output: WLC plot, elastica shape, reference charts
  - ZIP export with JSON + CSV of full results
